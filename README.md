## Démarer le projet

Copier et remplir le .env
~~~
cp .env.dist .env
~~~

~~~
.venv/Scripts/activate
pip install -r requirements.txt
streamlit run app.py
~~~

## Démarer l'API

Dans un autre terminal
~~~
cd api
.venv/Scripts/activate
pip install -r requirements.txt
python app.py
~~~
